Details

Developer API

This service is a JSON based REST API. You can access our API with the following Base-URL:

GET https://www.monitoor.com/api/

Authentication

User specific requests have to contain a parameter auth-token in the request header. This token can be retrieved by logging into your account. See the login documentation for details.

auth-token: xy10...EDW

Status Code and Error Handling

Any successfull call to the API will return status code 200. Any failed request will return a status code other than 200 and an error object in JSON format describing the error.
